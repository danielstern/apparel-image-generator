Image Generator API

Public Routes

/generate/:apparelType/:color/:logo[?format=...]

Generates a colorized image based on the provided criteria

Parameter Name Description Example
Apparel Type

The type of clothing or apparel to use as a base.

Valid options:

TSHIRT
hoodie
Color
Any color name, or the RGB color of the background image seperated by commas.
COMPENDIUM OF COLORS
0,0,255
6,72,24
PURPLE
sandybrown
Logo

The code of the logo appearing on the clothing.

Valid options:

CLEVER
sad
Query name Description Example
format The format of the image served http://localhost:7781/generate/tshirt/limegreen/CLEVER?format=JPEG
