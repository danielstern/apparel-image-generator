Image Generator API

Public Routes

/generate/:color/:logo

Generates a colorized image based on the provided criteria

Parameter Name Description Example
Color
Any color name, or the RGB color of the background image seperated by commas.
COMPENDIUM OF COLORS
0,0,255
6,72,24
purple
sandybrown
Logo

The code of the logo appearing on the clothing.

Valid options:

/generate/0,255,0/CLEVER
